# Preferências de Commit do Usuário

Este documento registra a preferência de fluxo de trabalho de Git/GitHub do proprietário deste projeto.

---

## 📌 Regra de Commit

> [!IMPORTANT]
> **NÃO gerar commits automáticos ou múltiplos commits pequenos sem autorização.**
> O assistente de IA deve manter as alterações locais salvas em arquivo, mas **apenas** realizar comandos de `git commit` ou `git push` quando o usuário der o comando explícito:
> 
> **"Gerar commit"** (ou similar).

---

## 📂 Escopo
Esta regra deve ser respeitada por todas as instâncias de IA que atuarem neste projeto e em futuros projetos relacionados.
