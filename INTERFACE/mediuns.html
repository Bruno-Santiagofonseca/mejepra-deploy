Como usar Médiuns

1
Cadastrar

Toque no botão + no canto inferior direito para adicionar um novo médium.

2
Editar ou Visualizar

Toque em qualquer médium da lista para abrir o cadastro completo e editar as informações.

3
Buscar

Use o campo de busca no topo para encontrar médiuns pelo nome rapidamente.

4
Status

Defina o status como Ativo, Afastado ou Fora da Casa.
